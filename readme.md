### 五个例子
**1.5 栈和自定义方法**
- 自定义集合类型都应该实现 `Len()` 和 `Cap()` 方法
- `*` 在不同位置含义不同
	- 在 `类型名称` 前 表示指针 *int
	- 在 两个数字或者变量之间表示乘法 x*y
	- 在 `变量名称` 前 表示解引用, 也就是引用该指针所指向的值 *stack
- 不建议使用不带返回值的return语句
